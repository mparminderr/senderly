##  senderly

# Follow steps to run the project


&ensp;1. After cloning repository and through terminal/cmd reaching in project directory, run ```npm install```


&ensp;2. Run ```
     npm run start
    ```

&ensp;3.Goto your browser and run http://localhost:3000/
